Palindromes: Palindromes are character sequences that read the same forward or backwards (e.g. the strings "mom" or "123 454 321"). This program uses recursive functions to determine whether a string is a character-unit palindrome, and also implements flags to indicate case sensitive comparisons and spaces.

Mountain Paths: This code reads elevation data and produces a file representing an image that, when visualized, displays each position on the map with a color that represents each elevation. By reading the data into a 2D vector, finding the min and max elevation, and computing the shade of gray for each cell in the map, I produced an output file. Using that output file, I then created a greedy algorithm to compute all paths (from the west to the east side of the terrain) that follow the policy of Ã¢ path of least resistance.

Bulls and Cows: This program works with arrays to implement the guessing game Bulls and Cows. The game works like this: when a number is guessed and a digit is in the correct location, it is a bull. When a digit is in the number, but in the incorrect location, then it is a cow. This code will help guess the entire number by informing the user how many bulls and cows are in their guesses. So if a four-digit code is 2894, and the user guesses 4698, then that is "1 bull and 2 cows" because '9' is in the correct location and '4' and '8' are in the code but are currently in the wrong location. The guessing continues until the code is guessed in the correct order (e.g. 4 bulls).

Grade Calculator: I used test-driven development, selection and iteration constructs, and functions to create a program that calculates my final grade in CSCE 121 according to the grading rubric in the syllabus.

Quadratic Equation: A simple program that inputs three coefficients (a, b, and c respectively) on a single line separated by spaces and then outputs the equation being solved along with each root on a seperate line.
